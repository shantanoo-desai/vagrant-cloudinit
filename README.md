# Vagrant Cloud-Init

## Setup on Majaro Linux

### Vagrant

```bash
sudo pamac install vagrant
```

version check using:

```bash
vagrant --version
Vagrant 2.2.19
```
### VirtualBox

Refer to Manjaro Linux [Wiki](https://wiki.manjaro.org/index.php?title=VirtualBox)

### `mkisofs`

```bash
sudo pamac install cdrtools
```

## Usage

```bash
vagrant up
```
once done

```bash
vagrant destroy
```
